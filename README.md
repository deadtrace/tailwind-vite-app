# tailwind-vite-app
Simple tailwind app built with Vite that uses:

- PostCSS (to power Tailwind)
- Autoprefixer (as a plugin to PostCSS)
- Tailwind (as a plugin to PostCSS)
- React (ReactDOM)
- Vite
- Github Actions (to validate HTML and deploy to Heroku)
- node-w3c-validator (to validate HTML)
- other useful packages

This app was created as a study task at ITMO University.

Deployed on Heroku:
https://tailwind-vite-app.herokuapp.com/
